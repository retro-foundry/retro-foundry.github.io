You’re now running a Jekyll blog on GitHub Pages. This is a sample post.

## Try it locally

1. Install Ruby and Bundler, then run:
   ```bash
   bundle install
   bundle exec jekyll serve
   ```
2. Open [http://localhost:4000](http://localhost:4000).

## Add new posts

Create a file in `_posts/` with the name format `YYYY-MM-DD-title.md` and add front matter:

```yaml
---
layout: post
title: "Your post title"
date: 2025-03-17 12:00:00 +0000
---
```

Push to the `main` branch and GitHub will build and publish your site at **https://retro-foundry.github.io**.
